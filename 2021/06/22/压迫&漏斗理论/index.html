压迫和漏斗理论

Katana
LevelDesign

发布于：2021年6月22日
次浏览

本文中，Qantm学院的游戏设计理论讲师卢克·麦克米兰介绍了如何在游戏设计中引导玩家，以经典的2D射击游戏为例，来说明其在澳大利亚昆士兰音乐学院的博士时所改编的作品中的实践。

作者：卢克·麦克米兰[2005.04.11]

现有环境驱动的RLD模型(Existing Models of RLD for Environment-Driven Titles)

术语

RLD：Rational Level Design,即理性关卡设计

就当代理性关卡设计的方法而言，视线这一关键的难度指标得到了广泛认可。也就是说，玩家视线越大，他们便越感觉自己能够掌控局势，并且越是特殊场景越容易发生。相反的情况也适用于限制玩家的视线，他们会变得更加犹豫不决。因为，这些类型的场景通常与更高难度有关。《毁灭战士3》中封闭空间和手电筒的使用是这些理论与原理发挥作用的一个很好例子。由于狭窄的走廊提供的视线有限，氛围变得非常紧张。并且由于视线有限，相关的电报时间有限，和敌人相遇时将会游戏难度将会困难。

压迫&漏斗背后的两个关键指标是入口(Portals)和遮挡物(Occluders)。根据镜头位置，视锥和视角的不同，入口会增加玩家的视线，而遮挡物会限制玩家的视线。图1摘自Byrne（2006）的《 Game Level Design》一书，从技术角度上很好地展示了这一概念。但是，这并不是我们为玩家呈现游戏世界的方式。

​ 图1

在几乎所有第一人称和一些第三人称视角游戏，镜头的兴趣点或者玩家操控角色，都会靠的非常近。图2是一个透视图的示例，并且是为游戏设计师提供的最适合的入口和遮挡物概念的示例。

​ 图2

尽管并非所有游戏都是3D的，但在当今的便携式和移动游戏市场中，2D游戏在商业上是可行的，并且很受欢迎。考虑到这一点，让我们从传统的自顶向下的2D游戏重新去考虑入口和遮挡物的概念。图中表示入口和遮挡物的局限性，由于游戏世界是通过使用屏幕上的玩家角色来进行抽象的，与图2中的示例不同，因此它不受屏幕上元素的限制。对于玩家角色驱动的2D游戏，入口便是屏幕本身，遮挡物则是屏幕物理上的框架。

​ 图3

在这一点，压迫&漏斗理论则可作为合适入口和遮挡物的替代方法。压迫&漏斗是一种理性的关卡设计原则，它着眼于如何在屏幕上，将诸如敌人，环境物体，弹幕射击和其他碰撞物体之类的元素，用于提高游戏难度并强烈激发玩家的情感。

[1]遮挡示例。转载自Game Level Design，第231页。Charles River Media Inc.

弗洛伊德是对的…

在继续探讨如何应用此原理的特定示例前，有必要解释下该理论的起源，以便将情感这一成分纳入我们的讨论范围。撇开玩笑和偷笑不谈，Dick被许多人认为是许多设计决策的灵感来源，但阴道却很少被赋予如此重要的设计意义，尤其是在游戏中。压迫和漏斗的理论源自Irem在1987年发布的R-Type(即异形战机式)游戏。R-type是一种传统的侧滚Shmup(“Shoot ‘Em Up”-全部射死)，具有丰富的美感，其很大程度上源于瑞士艺术家HR Giger的作品。在提及Giger的作品时，Irem在游戏设计中加入了一些有力元素，引起了玩家的强烈反响。

要了解这一点，我们需要考虑Shmup的工作原理，尤其是关卡中的Shmup。Shmup具有一个连贯的翻滚动作，不断迫使玩家进入更加紧密和激烈的范围，唯一的出路就是”直接通过”。在这一点上，R-Type类型游戏的动态符号学，和人类的诞生过程有着相似之处。弗洛伊德认为，出生时所带来的创伤，在后来所有的焦虑症中都起着重要作用，但是后来他放弃了这一理论。这并没有阻止后来的心理学家，例如奥托·兰克（Otto Rank）等人，探索出生过程及其对发育过程的影响。Grof（2002）讨论了生育的第二阶段，并将其描述如下：

在分娩的下一阶段，子宫的收缩持续侵袭着胎儿，但是子宫颈的扩张允许胎儿通过产道逐渐推进。这一阶段的重演，并不像前一阶段那样，涉及到对受苦受难者角色的专有认同；它还提供了进入被积压的凶残而又具有攻击性的巨大蓄水池“（译者：是指代羊水？）”的途径。。

不管出生时创伤的概念在心理学上是否重要，我们都可以从中提炼出两个非常重要的事情：收缩通常不会令人向往；其次，收缩时我们感到有必要同这些力量作斗争。就R-Type之类的Shmup而言，很容易看到相似之处-我们拥有一种可以迫使玩家进入狭窄空间的方法，解决这些问题并消除束缚的唯一方法，是摧毁并从屏幕上清除它们。最为重要的一点是，该论点还有很多超出本实践驱动部分范围的内容，但是，这些信息足以为以下实例提供背景支撑。

将理论付诸实践

既然我们对理论的起源有了一定的了解，那么现在该证明这种观念的实际含义了。为此，该特定作品将在Shmup风格的束缚下工作，考虑到随着移动游戏的出现，该风格的普及程度越来越高，这尤其合适。由于这是一种理性的设计方法，因此在继续进行之前，先确定一些关键指标很重要。

​ 图4

为了实现压迫和漏斗，我们需要了解，与玩家身上的各种接近矢量相关联的难度指标。再一次，我们以R-Type为例。在图4中，主轴线(Primary Axis)是主要射击轴线-弹幕离开玩家头像的轴线。通常，玩家会发现最容易对付的敌人，或是其他沿着正前方矢量移动的压迫元素，标记为绿色(Easy)。正对角线上的接近矢量为移动难度，最困难的接近矢量是与主轴相反的那些。（因为玩家通常无法用武器解决这些问题）。

一旦你了解了接近矢量，那么是时候了解如何压迫玩家了。基本上，游戏中的任何碰撞对象都可以作为一种压迫的代表。在Shmups的例子中，我们有几个非常明显的例子:

  1. 环境
  2. 弹幕
  3. 敌人

环境压迫

​ Table 1

从上面的示例中可以看出，漏斗区域是关卡设计中的一个区域，其目的是赋予玩家力量，既可创造针对令人费解的接近矢量的弹幕屏障，又可充当对敌人进行人群控制的机制。然而漏斗主要取决于两个可变因素：首先，漏斗区域与主轴方向一致，其次，漏斗区域的长度会影响战略上是否有利。图5和图6是这两种用途的很好的例子。

​ 图5

在图5的情况下，漏斗区域很短，这使玩家可以扩大视线，并确定他们通过漏斗区域时是否安全。在此示例中，漏斗区域可以防止，任何沿前方对角线的弹幕矢量击中玩家，并且由于视线较大，玩家能够更为有效地掌控局势。

​ 图6

在图6中，由于玩家占据了屏幕上的大部分空间，并限制了玩家的视线（类似由有限视线所关联的情绪），此时玩家将会更加犹豫于通过漏斗区域。因此，这不是一个漏斗，而是另一个环境的压迫点。在这两种情形下，设计人员始终注意到，玩家在所有的压迫点上进行“冲刺”，从而证明了，玩家会在尽可能的情况下，积极避免被压迫的观点。

敌人压迫

使用敌人作为压迫元素，与使用环境压迫略有不同，因为在几乎所有的Shmup中，玩家都可以使用武器早早地消灭这些压力，以防这些压迫元素变得过于强大。从RLD的角度来看，这意味着，使用更多”有机”的接近矢量，将敌人作为压迫元素，可能会更加广泛。由于玩家能够运用自己的火力，消灭这些压力，这使得我们对于弹幕的分析，会引发一点小问题。我们可以在表2中看到一个很好的示例。

​ Table 2

Table2是从联合国中队这款游戏中，提取的压迫时间顺序图。 在这个例子中，将有从玩家上下方，靠近玩家后方的接近矢量，它们的主轴朝向玩家，这很容易被玩家越过。Table2以有趣的方式，展示了与自动滚屏游戏相关的玩家心理。你会发现，大多数情况下，在横版滚屏游戏的关卡中，业余玩家总是会被迫紧靠屏幕左侧或底部。而在垂直滚屏中，敌人很少会从后方的接近矢量靠近，这与对玩家行为的相关观察和难度指标有关。不过，在联合国中队里，玩家可以在死亡前承受多次攻击，所以这个问题在某种程度上是可以避免的，但是仍应该非常谨慎地使用。回到当前例子，Table2展示了，如何使用多个独立移动的压迫元素，对玩家造成压制，并保持公平-当然也要记住，只要它们沿着主轴靠近，玩家也始终可以用武器来消灭这些压迫元素。

弹幕压迫

如今，弹幕设计经常被忽略，这主要是因为预先确定的子弹模式，在很大程度上与更加怀旧的游戏类型（即Shmup）有关。顺便说一句，BulletML是一个学习细微差别的弹幕设计有效系统，下面的示例均来自该特定脚本系统。

弹幕作为一种压迫形式很有趣，因为玩家通常可以通过，在敌人成为问题之前，消灭敌人来完全避免弹幕。因此，这使Shmups的RLD增加了新的复杂性，因为一旦玩家理解了特定敌人的弹幕类型可能造成的问题，他们将学会优先确定摧毁特定敌人的能力，但这超出了此特定讨论内容的范围。

在我们主要讨论压迫和漏斗时，弹幕设计是描述最终指标（扩张区域）重要性的有效方法。顾名思义，从情感和难度的角度来看，扩张与压迫相反，可以描述弹幕并赋予其难度等级，具体取决于弹幕所提供的扩张空间量。

​ 图7

图7是为垂直滚屏游戏的弹幕设计的示例，其中主轴面向屏幕顶部。在上面的示例中，弹幕具有一致的运动矢量，并且较大组中包含的每个弹丸具有相同速度。在此示例中，我们可以看到扩张可能（绿色箭头），远远超过红色的压迫箭头。这是导致玩家易于走位的情形。

​ 图8

图8是与压力（红色箭头）相比，由非常有限的扩张区域（绿色箭头），所创建的难度指标-一致的运动矢量和一致的速度。在Danmaku游戏中可以看到这种极端现象，其弹幕中有很多花样，创造了许多独特的接近矢量。通常，这些弹幕对于每个碰撞对象也将具有不同速度，这也导致情况变得更加复杂。

根据目前提供的信息，我们可以确定一些与压迫和漏斗相关的，非常基本的难度指标；

简单

  • 很多明显的扩张区域。
  • 压迫矢量有限。
  • 宽阔的漏斗区域。

困难

  • 独特或非常不明显的扩张区域。
  • 一次沿多个矢量压迫，尤其是与主轴相反的矢量。
  • 狭窄的漏斗区域。

总结思考

压迫和漏斗，是解决与2D关卡设计相关的设计问题的一种独特方法，但是它并不能代替视线理论，而是一种用来增强这种已经牢固模型的方法。这样，我们将压迫和漏斗应用到3D游戏世界中就不会太过困难，也或者与我们在本文中看到的内容不同，这只是证明该方法有效的下一个合乎逻辑的步骤。同任何事情一样，总有改进的余地。在此处，当我们将该理论应用于3D游戏世界的构建时，建筑理论和照明的惯例，显而易见的会是我们的兴趣点。
